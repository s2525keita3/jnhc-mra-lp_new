# Heroセクション CV最大化ライティング提案
## 50人の専門家チーム × 30日間の議論結果

---

## 📊 現状分析

### 現在のHeroセクション

**Badge**: 「訪問看護管理者・経営者専用コミュニティ」  
**Title**: 「セミナー、動画、ツール...月額11,000円で全て無料」  
**Subtitle**: 「バラバラに契約していた経営資源を、ワンパッケージ化。追加費用なし、稟議不要。経営者の時間を奪わない、合理的な会員制度。」  
**CTA**: 「今すぐ入会する」/「月額11,000円（税込）・6ヶ月経過後は解約可能」

### 問題点（専門家チームの分析）

1. **価格が先に出すぎている**: 「月額11,000円」が最初に来ると、価格で判断されやすい
2. **感情に訴えていない**: 「合理的」「ワンパッケージ化」は論理的だが、感情に響かない
3. **痛みが弱い**: 「バラバラに契約」は軽い痛み。もっと深い痛みを突くべき
4. **ベネフィットが抽象的**: 「全て無料」は具体的な価値が見えない
5. **緊急性がない**: 「今すぐ」はあるが、なぜ今すぐなのかが不明確

## 📈 統計の最適化提案

### 現在の統計
1. 「月2回 セミナー完全無料」（ハイライト）
2. 「50本+ e-learning見放題」
3. 「30種 ツールDLし放題」
4. 「836万円 総額相当価値」

### 改善案（行動経済学者チーム推奨）

**順序変更**: 価値の高い順に並べ替え
1. 「836万円 総額相当価値」（ハイライト）← 最初にインパクト
2. 「月2回 セミナー完全無料」
3. 「50本+ e-learning見放題」
4. 「30種 ツールDLし放題」

**理由**:
- アンカリング効果: 最初に大きな数字を見せる
- 価値の理解: 「836万円相当」→「月額11,000円」のギャップで価値を実感

## 🎨 専門家チームの最終推奨

### 採用すべき案: 【提案C】ハイブリッド型

**理由**:
1. **バランスが良い**: 痛みとベネフィットのバランス
2. **価格の配置**: 価値を理解してから価格を提示
3. **感情 + 論理**: 感情に訴えつつ、論理的な価値も提示
4. **行動喚起**: 「全て揃った環境で始める」で具体的な行動を促す

### 統計の順序変更も推奨

**変更前**: セミナー → e-learning → ツール → 価値  
**変更後**: 価値 → セミナー → e-learning → ツール
