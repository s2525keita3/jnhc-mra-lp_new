## 📍 RoiSection セクション（ROI計算）

### バッジ
```
ROI & Cost Comparison
```

### セクションタイトル（H2）
```
セミナー月2回参加するだけで元が取れる
```

**ライティング戦略**:
- 「月2回参加するだけで」という低いハードル設定
- 「元が取れる」という明確なROIメッセージ

### サブタイトル
```
他協会の隠れコスト vs 当協会の全込み価格
```

### 比較表

#### 他協会
```
年会費: 10,000円
セミナー: 5,000円×10回
---
年間総額: 60,000円
```

#### 当協会
```
月額: 10,000円
✓ セミナー完全無料
✓ e-learning見放題
✓ ツール30種DLし放題
---
年間総額: 120,000円
節約: 60,000円
```

**ライティング戦略**:
- 見た目は「120,000円 > 60,000円」だが、「節約: 60,000円」と表示
- これは「他協会で60,000円かかるものが、当協会では追加費用なし」という意味
- チェックマーク（✓）でポジティブな印象

### BREAK EVEN POINT ボックス
```
BREAK EVEN POINT
外部セミナー相場: 5,000円×2回 = 10,000円
月額10,000円の会費が、セミナー2回参加だけで回収可能
```

**ライティング戦略**:
- 英語の専門用語を使いながら、直後に日本語で説明
- 計算式で可視化
- 「回収可能」というポジティブな表現

### セミナー回収計算ボックス
```
セミナー月2回参加するだけで元が取れる計算:
外部セミナー相場
5,000円 × 2回 = 10,000円
月額10,000円の会費が、セミナー2回参加だけで回収可能
```
