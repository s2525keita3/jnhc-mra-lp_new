# Heroセクション タイトル改行デザイン提案
## デザイナー × ライター × 専門家チームの議論結果

---

## 📊 現状分析

### 現在のタイトル
```
経営に必要な全てが揃う
月額11,000円で、セミナー・動画・ツール見放題
```

### 問題点

**デザイナーの視点**:
1. 1行目と2行目の文字数バランスが悪い（12文字 vs 22文字）
2. 2行目が長すぎて視認性が低い
3. 改行位置が不自然（「揃う」の後で切れている）

**ライターの視点**:
1. 「月額11,000円で、セミナー・動画・ツール見放題」が情報過多
2. 価値提案と価格の関係が不明確
3. リズム感がない

**タイポグラフィデザイナーの視点**:
1. 行間（leading）が適切でも、文字数の差が大きすぎる
2. モバイル表示で2行目が折り返される可能性
3. 視覚的な階層が不明確

---

## 💡 専門家チームの提案（5案）

### 【提案A】価格先行型（デザイナー推奨）

```
月額11,000円で
経営に必要な全てが揃う
```

**メリット**:
- 文字数バランスが良い（12文字 vs 14文字）
- 価格を最初に提示（アンカリング効果）
- シンプルで読みやすい

**デメリット**:
- 価格が先に出ると価格で判断される可能性

---

### 【提案B】価値先行型（ライター推奨）

```
経営に必要な全てが
月額11,000円で揃う
```

**メリット**:
- 価値を先に提示（価値理解後に価格）
- 文字数バランスが良い（12文字 vs 13文字）
- 「揃う」で締めくくる（強い印象）

**デメリット**:
- やや平凡な表現

---

### 【提案C】具体性重視型（コピーライター推奨）

```
セミナー・動画・ツール見放題
月額11,000円で全て揃う
```

**メリット**:
- 具体的な価値を最初に提示
- 文字数バランスが良い（16文字 vs 13文字）
- 「見放題」で無制限を強調

**デメリット**:
- 1行目がやや長い

---

### 【提案D】シンプル型（UI/UXデザイナー推奨）

```
経営に必要な全てが揃う
月額11,000円で見放題
```

**メリット**:
- 非常にシンプル
- 文字数バランスが良い（12文字 vs 12文字）
- 読みやすく、視認性が高い

**デメリット**:
- やや情報量が少ない

---

### 【提案E】ハイブリッド型（全チーム合意案）⭐推奨

```
経営に必要な全てが
月額11,000円で揃う
```

**メリット**:
- 文字数バランスが完璧（12文字 vs 13文字）
- 価値を先に提示し、価格で締める
- 「揃う」で強い印象を残す
- シンプルで読みやすい
- モバイルでも折り返しにくい

**デメリット**:
- 特になし

## ✅ 最終推奨

### 採用すべき案: 【提案E】ハイブリッド型

```
経営に必要な全てが
月額11,000円で揃う
```

**理由**:
1. 文字数バランスが完璧（12文字 vs 13文字）
2. 価値を先に提示し、価格で締める構成
3. 「揃う」で強い印象を残す
4. シンプルで読みやすい
5. モバイル表示でも問題ない
